# sword-cli

## 项目简介
本项目为快捷脚手架工具，提供快捷的项目创建方式🚀🚀🚀


## 本项目相关命令

### 项目依赖安装
```
npm install
```
### 链接到本地npm
```
npm link
```

### 模板初始化
```
sword init 
```
