# sword-cli

## 项目简介
sword（ ⚔️ 利剑）
本项目为脚手架工程，提供快捷的项目创建方式，并给予项目工程能力 🚀🚀🚀


## 本项目相关命令

### 项目依赖安装
```
npm install
```
### 链接到本地npm
```
npm link
```

### 查看脚手架相关命令
```
sword 
```
